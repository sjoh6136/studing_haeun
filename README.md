# Lofi Study Hub

A simple and focused web application for study and relaxation, featuring lofi music and a clean interface.

## Features

- Lofi music player
- Minimalist study environment
- Integrated timer or history (based on `find_history.js` and `app.js`)

## Getting Started

To run the project locally, simply open `index.html` in your web browser.

## Deployment

This project is configured for deployment via GitHub Pages. Any changes pushed to the `main` branch will be automatically deployed.

## Built With

- HTML5
- CSS3
- JavaScript (Vanilla)
